# ClaudePanel

A retro-styled desktop HUD for Claude Code (Wails v3, Go backend + WebView frontend).

## Agent skills

### Issue tracker

Issues live as GitHub issues in `Eyalm321/claudepanel`, managed via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical triage roles, label strings equal their role names (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context — one `CONTEXT.md` + `docs/adr/` at the repo root, created lazily by `/grill-with-docs`. See `docs/agents/domain.md`.
